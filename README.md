# Levor Energy Website (Bootstrap + JS)

This is a static website starter for **levorenergy.com**.

## Structure
- `index.html` (Home)
- `about.html` (Company)
- `services.html` (Services)
- `projects.html` (Projects + filters)
- `media.html` (Gallery + video provision)
- `contact.html` (Contact + map + demo form)

## Run locally
Open `index.html` in your browser, or run a local server:

```bash
# Python
python -m http.server 8000
```

Then open: http://localhost:8000

## Add your video
Put an MP4 in `assets/video/` named `levor-intro.mp4` (or edit `media.html`).

## Deploy (Netlify)
- Drag-and-drop this folder in Netlify, **or**
- Push to GitHub and connect Netlify to the repo.
